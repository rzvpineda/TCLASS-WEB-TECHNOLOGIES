# Laboratory Activity 5
# Introduction to CSS

**Course:** Web Technologies

**Duration:** 3 Hours

---

# Title

Student Portfolio Website Styling

---

# Objectives

At the end of this activity, you should be able to:

- Create an HTML file base on the expected output
- Apply CSS to the HTML webpage.
- Create and link an external CSS file.
- Apply text, font, background, border, spacing, and layout properties.
- Improve the appearance of a webpage using CSS.

---

# Instructions

Create an HTML file base on the given sample.

Create an external stylesheet named **style.css**.

Link the stylesheet to your HTML file.

Complete the styling according to the specifications below.
